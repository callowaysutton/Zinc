{{DISPLAYTITLE:_SCREENICON (function)}}  

The _SCREENICON function returns -1 or 0 to indicate if the window has been minimized to an icon on the taskbar.


{{PageSyntax}}
::: minimized = '''_SCREENICON


{{PageDescription}}
* The function indicates -1 when the program is minimized to the task bar and 0 when not.
