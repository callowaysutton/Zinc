The '''&lt;&gt;''' condition symbol denotes that a value must not equal another value for the condition to be True. 


''Example usage:'' IF x [[Not_Equal|&lt;&gt;]] 320 THEN PRINT &quot;Not in center of screen&quot;


* Statements will evaluate as True or -1 when two values are not equal or False or 0 when equal. 
* More than one &lt;&gt; symbol in a numerical statement will create a [[Boolean]] evaluation of the ensuing numerical pairs.
