{{DISPLAYTITLE:_BLEND (function)}}
The {{KW|_BLEND (function)|_BLEND}} function returns enabled or disabled for the current window or a specified image handle when 32 bit.


{{PageSyntax}}
:''result%'' = {{KW|_BLEND (function)|_BLEND}}  [({{Parameter|imageHandle&amp;}})] 


{{PageDescription}}
* _BLEND returns -1 if enabled or 0 if disabled by {{KW|_DONTBLEND}} statement.
* '''Note: [[_DONTBLEND]] is faster than the default [[_BLEND]] unless you really need to use it in 32 bit!'''
