<h4 id="publish-your-project"><i class="icon-upload"></i> Publish your Project</h4>

<p>You can publish your project by pressing the <i class="icon-upload"></i> <strong>Publish</strong> button. This will automatically upload your project (If it is under 10 Mb and doesn’t use the INCLUDE command) to my website where you can view others codes and give advice. </p>

<p>There are a few rules for publishing: <br>
- No Unreasonable cursing. Cursing is allowed but you have to remember that younger kids will most likely see what you have said <br>
- No Nudity/18+ stuff unless a recognized artwork on WikiMedia <br>
- Read the community Code of Conduct for more rules</p>



<h4 id="publishing-to-other-websites"><i class="icon-upload"></i> Publishing to Other Websites</h4>

<p>If you want to, for some reason, build a project in Zinc for Ludum Dare, you can, as long as you state somewhere visible in the project that it was made using Zinc (This is only for if you are using the Drag and Drop based programming language)</p>
